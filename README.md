# fluid-simulation

## Usage
* `cd fluid-simulation/code`
* `mkdir build`
* `cd build`
* `cmake ..`
* `make`
* `./sim`

## Keyboard features
* Hold space key to see animation
* Use arrow keys to change perspectives
